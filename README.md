# Customer_Behavior_Analysis
## Cutomer_Behavior_Analysis using Python, SQL and PowerBI
### 📋Project Overview:
This project demonstrates an end-to-end data analytics workflow of Customer Behavior Shopping, starting from data loading and cleaning to advanced analysis and visualization. The goal is to extract meaningful insights from raw data and present them in a clear, actionable format using dashboards and reports.
### 📊Dataset:
Description: The dataset contains -

•	Customer demographics (Age, Gender, Location, Subscription Status) 

•	Purchase details (Item Purchased, Category, Purchase Amount, Season, Size, Color) 

•	Shopping behavior (Discount Applied, Promo Code Used, Previous Purchases, Frequency of Purchases, Review Rating, Shipping Type) 

•	Format: Excel.

### 🛠️Tools Used:
•	Python (Pandas)

•	SQL (PostgreSQL)

•	Power BI

•	Gamma (for presentation)

### 📌Steps:
1.	Data Loading:
   
          o	Imported dataset using Python.
  
          o	Performed initial inspection of structure and data types.
  
2.	Exploratory Data Analysis (EDA):
   
          o	Analyzed distributions, trends, and relationships.
  
          o	Identified missing values and outliers.
  
3.	Data Cleaning:
   
          o	Handled missing values.
  
          o	Removed duplicates and corrected inconsistencies.
  
          o	Standardized formats and data types.
  
4.	SQL Analysis:
   
          o	Loaded cleaned data into relational databases.
  
          o	Wrote queries for aggregation, filtering, and joins.
  
          o	Extracted insights using PostgreSQL.
  
5.	Dashboard Creation:
   
          o	Built interactive dashboards in Power BI.
  
          o	Included KPIs, charts and filters for user interaction.
  
6.	Reporting & Presentation:
   
          o	Created a detailed analytical report.
  
          o	Developed a presentation using Gamma to communicate findings.
  
### 📈Dashboard:

•	Interactive Power BI dashboard showcasing key metrics.

•	Includes visualizations such as:

  o	Trend analysis
  
  o	Category-wise breakdown
  
  o	KPI indicators.
  
### 🔍 How to Run:

1.	Clone the repository.
  
2.	Install required Python libraries: pip install pandas.
	
3.	Run the Python scripts for data loading and analysis.
	
4.	Execute SQL scripts in your preferred database (PostgreSQL).
	
5.	Open the Power BI file to explore the dashboard.
  
6.	Review the report and presentation for insights.

•	Identified key patterns and trends in the dataset.

•	Provided actionable insights for decision-making.
